movi-api
Desenvolvida API RESTful para possibilitar a leitura da lista de indicados e vencedores da categoria Pior Filme do Golden Raspberry Awards.

Como usar
Para execução do projeto, é necessário instalação do Java17.

Endpoints desenvolvidos
/movie-api/movie/all -> Para listar todos filmes inseridos
/movie-api/producer/with-winner -> Para identificar os produtores com filmes premiados
/movie-api/producer/winner -> Para listar os produtores com filmes premiados, conforme orientação da documentação recebida

Testes
Criada classe de Teste UtilsTest.java.
